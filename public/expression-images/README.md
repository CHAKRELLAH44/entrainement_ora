Pour ajouter des photos à la fonctionnalité "Exprimer", placez vos images dans ce dossier :

📁 `/public/expression-images/`

Noms de fichiers requis :
- `image-1.jpg` (ou .png)
- `image-2.jpg` (ou .png)
- `image-3.jpg` (ou .png)

Les images seront automatiquement chargées par l'application. Assurez-vous que :
- Les fichiers sont au format JPG, PNG ou WebP
- La taille optimale est de 800x600px environ
- Le poids ne dépasse pas 2MB par image

Une fois les images ajoutées, redémarrez l'application pour les voir.
